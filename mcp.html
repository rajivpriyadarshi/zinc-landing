Zinc MCP Server

Give your AI agents direct access to payment infrastructure through MCP. Intent-based card provisioning with policy enforcement—connect any MCP-compatible agent in minutes.

Server URL: https://mcp.zinc.dev/api/mcp

How It Works

1

Declare Intent

Agent specifies purchase purpose, expected amount, and merchant constraints

2

Policy Evaluation

Zinc validates the intent against your configured spend policies

3

Card Provisioned

A locked virtual card is issued specifically for this intent

4

Transaction Verified

Payment executed, receipt captured, and full audit trail logged

Get authenticated user profile

Read current spend policy

zinc.intents.list
List your intents

Get intent details including card info

List cards for the authenticated user
zinc.cards.get
Get card details by ID

Unfreeze a card
zinc.cards.rotate
Rotate card credentials without changing policy
zinc.cards.close
Close a card permanently

→ Transactions

zinc.transactions.list
List transactions for a specific card
zinc.transactions.get
Get transaction details

Example Flows

Create Intent for Purchase

Declare a purchase intent that auto-approves within policy and issues a locked card

Request
zinc.intents.create({
  purpose: "Buy API credits for OpenAI",
  expectedAmount: 5000,  // cents
  merchant: "openai.com"
})
Response
{
  intentId: "int_abc123",
  status: "card_issued",
  card: {
    cardId: "card_xyz",
    last4: "4242"
  },
  approvalRequired: false,
  advisories: []
}

Retrieve Card Details for Payment

Fetch sensitive card information (PAN) after intent approval to complete purchase

Request
zinc.cards.get_sensitive({
  cardId: "card_xyz",
  intentId: "int_abc123",
  reason: "Making purchase for OpenAI API credits"
})
Response
{
  cardId: "card_xyz",
  pan: "4242424242424242",
  expMonth: 12,
  expYear: 2026,
  cvv: "123",
  billingAddress: { ... }
}

Recurring Subscription Setup

Create recurring intent with auto-unlock per billing window for automated monthly charges

Request
zinc.intents.create({
  purpose: "Figma subscription",
  cardId: "card_existing_123",
  expectedAmount: 1200,
  expectedMerchant: "Figma",
  recurring: {
    cadence: { type: "monthly" },
    window: {
      startDaysBefore: 3,
      endDaysAfter: 2
    },
    match: {
      amountTolerance: 0.1,
      merchantMatch: true
    }
  }
})
Response
{
  intentId: "int_recurring_456",
  status: "active",
  recurring: {
    nextWindow: "2024-02-28",
    cadence: "monthly"
  },
  approvalRequired: false
}

Quick Setup

Claude Code
Cursor
Manual Config
# Add Zinc MCP server to Claude Code
claude mcp add zinc --transport http --url "https://mcp.zinc.dev/api/mcp"

# Authenticate with your API key
claude mcp auth zinc --token "your_api_key"

# Verify connection
claude mcp test zinc
# Add to Cursor settings (Settings > MCP Servers)
{
  "zinc": {
    "transport": "http",
    "url": "https://mcp.zinc.dev/api/mcp",
    "headers": {
      "Authorization": "Bearer YOUR_API_KEY"
    }
  }
}
// Add to your MCP configuration file
{
  "mcpServers": {
    "zinc": {
      "transport": "http",
      "url": "https://mcp.zinc.dev/api/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
